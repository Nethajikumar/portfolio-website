# 🪐 Premium Portfolio | Nethaji K - Full Stack Architect

A high-performance, cinematic personal portfolio designed for impact. Built with modern web technologies, this project features advanced UI/UX patterns, fluid animations, and a focus on high-end digital aesthetics.

---

## ✨ Key Features

### 🎨 High-End UI/UX
- **Interactive Ambient Glows**: A hardware-accelerated, cursor-following background spotlight for a dynamic feel.
- **Bento Grid Architecture**: A modern, structured "Technical Ecosystem" grid showcasing skills with premium spotlight effects.
- **Scroll-Reactive Backgrounds**: Atmosphere that subtly shifts and transforms based on user scroll progress.
- **Premium Project Showreel**: Sophisticated project gallery with smooth modal transitions and detailed feature breakdown.

### ⚡ Performance & Tech
- **Core Stacks**: [Vite](https://vitejs.dev/) + [React](https://react.dev/) + [Tailwind CSS](https://tailwindcss.com/)
- **Animation Orchestration**: [Framer Motion](https://www.framer.com/motion/) for staggered reveals, page transitions, and interactive physics.
- **SEO Ready**: Deeply integrated [React Helmet Async](https://github.com/staylor/react-helmet-async) for per-page meta-tags and optimized link previews.
- **Micro-Interactions**: Features like one-tap "Email Copy-to-Clipboard" with stylized toast notifications via `react-hot-toast`.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation & Run
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Nethajikumar/portfolio-website.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
